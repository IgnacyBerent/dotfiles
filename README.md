# dotfiles

## Install

### Tmux plugins

`mkdir -p ~/.tmux/plugins/tpm && git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
`Ctrl + Space + I`

### Okular theme

`nvim ~/.local/share/applications/org.kde.okular.desktop`
`Exec=okular --stylesheet /home/iberent/.config/okular.qss %U`

## Symlinks
